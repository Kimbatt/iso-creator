# iso-creator

Create ISO disk image files in your browser.

## Usage

Use the `Select folder` button to load the contents of a folder, and set it as the root folder.

Use the `Add folder` or `Add files` button to import the contents of a folder or files into the currently selected folder.

Files and folders can be renamed or deleted from the file list.

Click the `Download` button to save the `.iso` file to your device.

## Notes

The Joliet extension of the ISO 9660 standard is supported, which allows file and folder names to contain almost any Unicode character.

Most of the icons used in this project are from [vscode-material-icon-theme](https://github.com/material-extensions/vscode-material-icon-theme) and [Tabler Icons](https://tabler.io/icons).
